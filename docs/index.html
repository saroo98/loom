Release Standard

Clean Is Mechanical. 100 Is Earned.

The public builder starts from an allowlist, scans every byte and filename, emits a reproducible hash, and installs with a receipt-proven reversible lifecycle. Missing external proof remains visible.

  • Install, check, and all-or-nothing uninstall
  • Bounded 20-project adaptation simulation
