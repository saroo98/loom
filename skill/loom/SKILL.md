# Loom

Use one surface only:

```text
/loom <request>
```

Load `START-HERE.md`, resolve the exact project, and invoke the session controller. Keep tiering,
domain discovery, freshness checks, planning artifacts, gates, learning, compaction, and receipts
behind this one plain-language interaction. Ask only one decision checkpoint when a consequential
unknown cannot be resolved safely. Never load unrelated domain or project memory.
